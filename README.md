Objetivo
Este projeto visa eliminar a "cegueira operacional" em processos logísticos, automatizando a análise de estoque parado e perdas de qualidade. A solução transforma dados brutos em insights financeiros, permitindo que a gestão identifique rapidamente onde o capital está imobilizado e quais produtos estão impactando negativamente o resultado.

🚀 Funcionalidades
- Limpeza Automática: Padronização de datas e tratamento de dados faltantes.

- Análise Financeira: Cálculo preciso do custo de capital imobilizado por item.

- Diagnóstico de Pareto: Identificação dos "vilões" de estoque e reprovação.

- Visualização Estratégica: Dashboards automatizados para suporte à tomada de decisão.

🛠️ Tecnologias Utilizadas
- Linguagem: Python

- Manipulação de Dados: Pandas

- Visualização: Matplotlib

- Ambiente: VS Code

Como Executar
Clone este repositório:

Bash
git clone https://github.com/dpelentil-ops/otimizacao-logistica-python.git
Instale as dependências:

Bash
pip install -r requirements.txt
(Caso não tenha o arquivo, instale manualmente: pip install pandas matplotlib)

Coloque seu arquivo dados_logística_exemplo.csv na pasta raiz e execute:

Bash
python analise_logistica.py
